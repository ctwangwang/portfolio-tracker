# Portfolio Tracker

A personal asset tracking application with multi-currency support for stocks and cash management.

## Features

- **Multi-Market Stock Tracking**: Support for US, Hong Kong, and Taiwan markets
- **Real-time Price Updates**: Stock prices refresh every minute
- **Multi-Currency Support**: Track assets in USD, CAD, HKD, TWD, CNY and more
- **Cash Management**: Track cash holdings in different currencies
- **Privacy Mode**: Hide/show portfolio values
- **Responsive Design**: Works on desktop and mobile devices
